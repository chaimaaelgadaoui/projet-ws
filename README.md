LAB9 de Gestion des Étudiants (Android & PHP/MySQL)

🚀 Fonctionnalités
•Affichage en temps réel : Liste des étudiants récupérée depuis une base de données MySQL.
•Ajout d'étudiants : Formulaire pour saisir le nom, prénom, ville et sexe.
•Modification : Mise à jour des informations d'un étudiant via une boîte de dialogue (Dialog).
•Suppression : Suppression d'un étudiant avec confirmation.
•Interface intuitive : Utilisation de ListView avec un Adapter personnalisé et des composants Material Design.

🛠 Technologies utilisées

Mobile (Android)
•Java : Langage de programmation principal.
•Volley : Bibliothèque pour effectuer des requêtes HTTP (POST/GET).
•Gson : Pour convertir les réponses JSON du serveur en objets Java (POJO).
•Material Design : Pour les composants d'interface utilisateur (Dialogs, Buttons, etc.).
Backend (Web Service)
•PHP : Langage script côté serveur pour traiter les requêtes.
•MySQL : Base de données pour stocker les informations des étudiants.
•JSON : Format d'échange de données entre le mobile et le serveur.
